# Калькулятор
## Калькулятор  с множеством возможностей

## Особенности и возможности
- Базовые опирации включая извлечение квадратного корня и возведение в квадрат
- История вычислений
- Построение графиков
- Красивый и современный интерфейс сделанный с помощью библиотеки customtkinter

## Технологии
- Python 3.12
- customtkinter
- matplotlib
- PIL

## Запуск
```bash
pip install requirements.txt
```
```bash
python app.py
```
